krkn Dashboard is the visualization component of [krkn-hub](https://github.com/redhat-chaos/krkn-hub/tree/main). It offers a set of user-friendly web interfaces through which users can manipulate and observe Chaos experiments.

# Run Options
You can run the Krkn Dashboard using either of the following methods:

- Standalone Mode
Follow the installation steps here:
https://github.com/krkn-chaos/krkn-dashboard/tree/main/docs/installation.md

- Containerized Version
Build and run the container image using the guide here:
https://github.com/krkn-chaos/krkn-dashboard/tree/main/containers/build_own_image-README.md
